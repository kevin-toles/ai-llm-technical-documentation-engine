## Key Design Principles

| Principle | Description |
|-----------|-------------|
| **Per-Chapter Processing** | Workflow executes for EACH chapter in the source book |
| **Source-First** | LLM reviews source chapter BEFORE taxonomy |
| **Metadata-Driven** | Concept matching uses enriched metadata, not raw text |
| **No Arbitrary Limits** | LLM requests ALL relevant chapters |
| **Tier-Aware Synthesis** | Annotations structured by tier priority (from taxonomy) |
| **Validation Required** | LLM must verify genuine relevance, not just keyword overlap |
| **Gap Acknowledgment** | Missing tier coverage is explicitly noted |
| **Citation Precision** | Chicago-style citations with page numbers |
